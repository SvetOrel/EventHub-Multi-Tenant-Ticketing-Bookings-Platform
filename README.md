# 🎟️ EventHub – Backend Developer Portfolio Project

**EventHub** is a production-style backend project that simulates a real-world ticketing & event management platform.  
It is designed to cover **most backend roadmap topics**: APIs, databases, caching, queues, observability, CI/CD, and cloud deployment.

---

## 🚀 Tech Stack
- **Language/Framework**: Java 21 + Spring Boot 3
- **Database**: PostgreSQL
- **Caching**: Redis
- **Messaging**: RabbitMQ
- **Storage**: MinIO (S3-compatible)
- **Email testing**: MailHog
- **Infrastructure**: Docker + Docker Compose
- **CI/CD**: GitHub Actions
- **Documentation**: OpenAPI (Swagger)
